# Python
---

- 生成 requirements.txt 文件
   ```bash
   pip freeze > requirements.txt
   ```

- 安装 requirements.txt 文件
   ```bash
   pip install -r requirements.txt
   ```

- 升级 pip
   ```bash
   pip install --upgrade pip
   ```

- 升级包
   ```bash
   pip install --upgrade <package>
   ```

- 卸载包
   ```bash
   pip uninstall <package>
   ```

- 列出已安装的包
   ```bash
   pip list
   ```

- 列出已安装的包及其版本
   ```bash
   pip list --format=freeze
   ```
